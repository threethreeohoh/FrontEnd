# FrontEnd

# 사용한 기술 스택 
- React (Javascript)
- Axios
- Node.js
- VSCode

# 브랜치 전략
- Git Flow
  - main branch : 테스트 완료 후 실제 배포되는 브랜치
  - develop branch : 개발 브랜치
  - feature : 기능 브랜치
 
# 배포 환경
- AWS
  - S3스토리지
  - ECR(Elastic Container Registry)
  - EC2
  - RDS

# 배포 도구
- Docker
- Nginx

# CI 구축
- Github Actions
